Zen Coding for <textarea>

This is a demonstration of Zen Coding for <textarea> plugin. To use it on your web-page, simply incude zen_textarea.js on your page.

Click on “Powered by Zen Coding” icon to see available keyboard shorcuts.

This plugin also supports expading abbreviations/indenting by Tab key and formatted line break (add indentation of previous line when hitting Enter key). These options are off by default, but you can turn them on globally or locally (for specified textarea). For a list of available options see default_options variable of manager.js file.

To enable option globally just call zen_textarea.setup() method passing object with options and their values:

zen_textarea.setup({pretty_break: true, use_tab: true});
To change options for specified textarea, add zc-{option_name}-{value} class name for <textarea> tag: 
<textarea class="zc-syntax-xsl zc-use_tab-true zc-pretty_break-false"></textarea>
